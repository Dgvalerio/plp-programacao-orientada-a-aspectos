## Atividade consiste em:
Importe o projeto eclipse presente no arquivo "labAOP.zip".\
Em seguida, execute o roteiro de laboratório apresentado no PDF anexo ("roteiro-labAspectJ.pdf").

**Entrega:**

O programa precisa ser apresentado e seu código-fonte explicado. A explicação pode ser:
- Presencial: mediante agendamento com o professor
- Via google meet: mediante agendamento com o professor
- Por vídeo:
  - Desde que o rosto do apresentador seja visível durante a apresentação. O rosto pode ser um quadro pequeno do vídeo.
  - Envie apenas o link para o vídeo, não anexe o arquivo do vídeo.
